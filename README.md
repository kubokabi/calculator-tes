# Hagematech Calculator

Hagematech Calculator is a modern scientific calculator Web Component that can be embedded into any HTML page using a simple custom tag.

It is designed for CDN usage, supports Shadow DOM styling isolation, scientific functions, memory operations, custom colors, gradients, light/dark mode, keyboard support, and a bilingual guide book.

## Features

- Easy CDN installation
- Simple custom element usage
- Shadow DOM isolated style
- Scientific calculator layout
- Physical calculator style UI
- Responsive design
- Custom gradient themes
- Custom hexadecimal color support
- Light and dark mode
- DEG/RAD angle mode
- Memory functions: MC, MR, M+, M-, MS
- Basic operations: addition, subtraction, multiplication, division, percent
- Scientific functions: sin, cos, tan, asin, acos, atan, log, ln, square root, cube root, power, factorial, inverse
- Constants: π and e
- Keyboard support
- Bilingual guide book: English and Indonesian
- Powered by math.js

## Installation

### CDN

Use the CDN script before the closing `</body>` tag or inside the `<head>` with `defer`.

```html
<script defer src="https://cdn.jsdelivr.net/gh/kubokabi/calculator-tes@1.0.1/hagematech-calc.js"></script>
